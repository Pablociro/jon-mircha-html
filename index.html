Etiquetas semanticas estructurales

Div-es una etiqueta contenedora que no tiene semantica
header-cabecera de un sitio web o de una seccion
Main-definde la sección principal del documento, sólo puede existir una etiqueta main por documento
Footer- Pie de página de un sitio web o de una sección
nav-representa una navegación, conjunto de enlace que formen un menu d enavegaciuon o tambien podria ser una botonera
Article-Define un bloque de contenido autónomo que puede existir en cualquier contexto. Puede tener su propio encabezado, pie de página, secciones... Útil para una lista de publicaciones de blog.
Aside-Define un bloque de contenido relacionado con el contenido principal. Generalmente se muestra como una barra lateral.
Section-Define una sección de contenido dentro de una página web.
Adrdress-Define un bloque para la información de contacto , por ej una direccion fisica , de contactos etc
☝️

Etiquetas de bloque vs etiquetas de linea

div-Define un bloque genérico de contenido, que no tiene ningún valor semántico. Se utiliza para elementos de diseño como contenedores, genera saltos de linea con sus etiqueta hermanas, ocupa tood el espacio de ancho.
span-Define un contenedor genérico de contenido en línea , que no tiene ningún valor semántico, solo ocupa el espacio necesario que tiene su contenido. soy otra span ☝️
